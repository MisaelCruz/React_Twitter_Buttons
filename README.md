# Componentes de Twitter con React

Una práctica de desarrollo Frontend enfocada en la creación, modularidad y manejo de estados de componentes reutilizables en React. El proyecto replica la interacción de las tarjetas y botones de Twitter (como las acciones de seguir y el diseño del perfil).

### 🛠️ Tecnologías utilizadas:
* **React**
* **JavaScript**
* **Vite** (Herramienta de compilación rápida)
* **Bootstrap**
